## Rules

- If a message gets replies, keep testing it.
- If a message gets praise but no next step, treat it as weak.
- If a feature request does not clearly unblock message clarity, onboarding, demo quality, or pilot conversion, log it and deprioritize it.
- If the same pain appears 3+ times from the right ICP, move it into core messaging.
