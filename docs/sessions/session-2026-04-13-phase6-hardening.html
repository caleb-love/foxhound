3. Code changes shipped

API scope enforcement

apps/api/src/plugins/auth.ts

  • Added API key scope parsing
  • Added prompt route scope checks
  • GET /v1/prompts* requires prompts:read
  • Non-GET prompt routes require prompts:write
  • Legacy keys with null scopes still retain full access

Prompt route hardening

apps/api/src/routes/prompts.ts

  • Added route-specific rate limits
  • Catches Postgres unique violation 23505 for duplicate prompt creation
